정확하고 강력한 음성 인식 STT Speech To Text

마인즈랩의 STT(Speech To Text) 엔진은 음성을 텍스트로 빠르고 정확하게 변환하는 음성 인식 엔진입니다.
딥러닝 방식의 인공 신경망(DNN)이 적용된 마인즈랩의 STT는 한국어와 영어에 특화된 강력한 음성인식 성능을 자랑합니다.
실시간으로 사용자의 음성을 텍스트로 변환하거나, 기존 음성 파일을 한번에 텍스트로 변환하여 다양한 비즈니스에 적용할 수 있습니다.

Process

features

정확한 한국어 인식
마인즈랩의 STT는 한국어에 특화된 언어 및 단어 모델 기반의 학습을 마쳤습니다. 각 나라의 언어에 맞도록 별도로 설정해야 하는 기존 글로벌 엔진 대비 더 높은 정확도와 인식률을 자랑합니다.
강력한 딥러닝 활용
국내 최고 수준의 강력한 DNN(인공신경망)을 활용한 마인즈랩의 STT는 화자의 발성 파형에서 특징을 추출하여 가장 유사한 텍스트로 정확하게 변환합니다.
추가 학습을 통한 커스터마이징
기본적으로 1,000시간의 음성 데이터 학습을 마친 마인즈랩의 STT는 각 산업 영역 고유의 음성 데이터를 바탕으로 추가 학습이 가능합니다. 추가 음성 학습을 마친 마인즈랩의 STT를 통해 산업 특성에 맞게 음성인식 기능을 커스터마이징해보세요
소음 환경에서도 정확한 인식
한국전자통신연구원의 고속 잡음 전처리 기술이 적용된 마인즈랩 STT는 주변의 소음이 심하더라도 정확하게 음성을 인식합니다.
실시간 텍스트 변환
마인즈랩의 STT 엔진은 실시간으로 텍스트 결과를 스트리밍할 수 있습니다. API 호출로 인한 결과가 실시간으로 스트리밍되거나, 저장된 녹취 파일을 실시간으로 변화하여 API 형태로 호출됩니다.
높은 확장성과 활용성
다양한 기기에서 작동하는 마인즈랩의 STT 엔진은 유통, 물류, IT 등 모든 분야에서 활용할 수 있습니다. 자동 음성 인식 시스템을 비롯하여 음성 검색은 물론 음성 데이터 분석 등 심층적인 목적으로도 사용할 수 있습니다.
