IR QA(Information Retrieval Question Answering)

마인즈랩의 IR QA는 다양한 분야의 지식 소통이 가능한 인공지능 소프트웨어입니다

준비사항

① Input: 텍스트(질문)

② 아래 Model 중 택 1

  • 위키/백과 (default)
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/api/irqa/

② Request 파라미터 설명

키 설명 타입
ID 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
key 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
cmd 어떤 API 를 사용할지에 대한 간단한 키워드 (runIRQA) string
question 답을 요구하는 질문 (텍스트) string

③ Request 예제

curl -X POST \
https://api.maum.ai/api/irqa/ \
-H 'content-type: multipart/form-data;
boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F ID=(*ID 요청 필요) \
-F key=(*key 요청 필요) \
-F cmd=runIRQA \
-F 'question=김연아의 키는 몇이야?'
