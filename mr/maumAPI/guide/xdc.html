XDC(eXplainable Document Classifier)

마인즈랩의 XDC(문서 분류, eXplainable Document Classifier)는 내용과 분량에 관계 없이 어떤 문서든 즉시 분류한 뒤 분류 이유를 설명할 수 있는 혁신적인 인공지능 문서 분류 기술입니다.

준비사항

① Input: 본믄 (텍스트)

② 아래 Model 중 택 1

  • 뉴스 분류 (default)
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/api/xdc/

② Request 파라미터 설명

키 설명 type
ID 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
key 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
cmd 어떤 API 를 사용할지에 대한 간단한 키워드 (runXDC) string
text 분류를 수행할 본문 (텍스트) string

③ Request 예제

curl -X POST \
https://api.maum.ai/api/xdc/ \
-H 'content-type: multipart/form-data;
boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F ID=(*ID 요청 필요) \
-F key=(*key 요청 필요) \
-F cmd=runXDC \
-F 'text=문재인 대통령이 자영업자 지원 대책 마련을 지시했다.
자영업자 지원 대책을 통해 민생 경제 살리기에 나서겠다는 의도다. '

④ Response 파라미터 설명

Response
키 설명 타입
data API 결과 data (json format) object
status API 동작여부 (Success / Fail) string
data: API 결과
키 설명 타입
sent attns 문장별 분석 결과 (json list format) list
cls 분류 결과 (json list format) list
top_word_indices_lists 문장 내 “Top word” list
* Top word: 분류 결과에 영향을 크게 미친 단어		 list
sentences 본문 원문 문장의 나열 list
top_sent_indics 본문 내 “Top sentence” list
* Top sentence: 분류 결과에 영향을 크게 미친 문장		 list
sent attns: 문장별 분석 결과
키 설명 타입
weight 문장의 영향도 (0~1) number
word attns 단어별 분석 결과 (json list format) list
word attns: 단어별 분석 결과
키 설명 타입
end 단어의 끝(n) int
start 단어의 시작(n) int
weight 단어의 영향도 (0~1) number
cls: 분류 결과
키 설명 타입
label 분류 결과의 라벨 (확률 높은 순대로 배열) string
probability 분류 결과 라벨에 대한 확률(0~1) number
