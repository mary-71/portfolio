개발 환경 세팅

2) ‘이름’과 ‘이메일’을 기입하여 hello@mindslab.ai 메일을 보냅니다.

3) 담당자와 협의 후 담당자가 이메일을 통해 ID와 키를 발급해줍니다.

TTS (Text-To-Speech)

마인즈랩의 TTS API는 텍스트 문장 또는 파일을 음성으로 변환시켜주는 API입니다. 20시간 학습만으로도 다양한 모델로 자연스럽게 음성을 합성해주는 마인즈랩의 API를 경험해보세요.

준비사항

① Input: 텍스트(문장) - Output: 음성파일 (format) *제약사항: 100자 이내의 문장(한글/영어 기준)

② 아래 Model 중 택 1

  • 한국어 - 여성 성우
  • 영어 - 여성 성우
  • 한국어 - 캐릭터 목소리 (뽀로로)
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/api/tts/

② Request 파라미터 설명

키 설명 type
ID 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
key 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
cmd 어떤 API 를 사용할지에 대한 간단한 키워드 (runFileTts) string
lang 사용할 모델의 언어 (ko_KR / en_US) string
sampling 문장 (*100자 이내) string

③ Request 예제

curl -o sample.wav -X POST https://api.maum.ai/api/tts/ -H
'Content-Type: application/json' -d '{"ID":"ID 요청 필요",
"key": "key 요청 필요", "cmd": "runFileTts",
"lang":"0","text":"안녕하세요, 마인즈랩입니다."}'
